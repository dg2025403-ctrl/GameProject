## 📚 데이터 출처 (Data Source)

본 프로젝트는 공개 데이터셋인 **Video Game Sales Dataset**을 활용하여 제작되었습니다.

해당 데이터는 전 세계 비디오 게임 판매량 정보를 수집한 데이터로, 약 **16,500개의 게임 데이터**를 포함하고 있습니다.

### 🔗 데이터 정보

| 항목 | 내용 |
|------|------|
| 데이터명 | Video Game Sales Dataset |
| 제공 플랫폼 | Kaggle |
| 데이터 제공자 | Gregorut |
| 원본 데이터 | VGChartz |
| 데이터 개수 | 약 16,500개 |
| 사용 파일 | `vgsales.csv` |

### 📌 사용 변수

| 변수명 | 설명 |
|--------|------|
| Name | 게임 이름 |
| Platform | 출시 플랫폼 |
| Year | 출시 연도 |
| Genre | 게임 장르 |
| Publisher | 제작사 |
| NA_Sales | 북미 판매량 (백만 장) |
| EU_Sales | 유럽 판매량 (백만 장) |
| JP_Sales | 일본 판매량 (백만 장) |
| Other_Sales | 기타 지역 판매량 (백만 장) |
| Global_Sales | 전 세계 판매량 (백만 장) |
